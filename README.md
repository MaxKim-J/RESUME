### 자기소개

대학생 자바스크립트 개발자로 React와 Vue를 사용한 웹 클라이언트 외주 개발, 개인 프로젝트, 엔지니어 10명 규모의 스타트업에서 개발자 인턴을 경험했습니다. 현재 패션 프리오더 서비스 스타트업 [모예](https://moye.kr/)에서 프론트엔드 개발자로 재직중입니다.

기술의 원리를 탐구하며 공부하는 것을 좋아하고 기록을 남기는 꾸준한 공부를 지향합니다. [TIL](https://github.com/MaxKim-J/TIL), [ALGO](https://github.com/MaxKim-J/Algo), [튜토리얼](https://github.com/max-kim-tutorial) 레포지토리, 개인 [기술 블로그](https://maxkim-j.github.io/)에 기록을 남기며 공부하고 있습니다. 무언가를 잘 이해하고 있다면 간단하게 설명할 수 있어야 한다는 생각으로 이해한 바를 글로 설명하는 연습을 계속하고 있습니다. 

chrome 확장 앱 [외대 종강시계](https://chrome.google.com/webstore/detail/%EC%99%B8%EB%8C%80-%EC%A2%85%EA%B0%95%EC%8B%9C%EA%B3%84/jadlpknbgnmmelikpcaogikohieafaem?hl=ko)(2020년 11월 기준 유저 265명)와 법원 종합법률정보 사이트의 대법원 판례를 자동으로 트윗해주는 [판례요지봇](https://twitter.com/precedent_bot)(2020년 11월 기준 팔로워 234명)서비스를 기획하고 빌드, 운영하고 있습니다. 개인적으로 개발한 서비스들을 사용하는 유저들을 보며 보람을 느끼고,  항상 유저의 입장에서 앱에 바라는 것이 무엇일지 생각해보려고 노력합니다. 

또한 글로벌 솔루션 스타트업 [TravelFlan](https://www.travelflan.com/)의 프론트엔드 팀에서 인턴으로 6개월간 근무하며 디자이너, PM, Django 백엔드 개발자, 해외 개발자들과의 협업과 적극적인 코드 리뷰를 경험했습니다. 인턴십에서 쌓은 경험을 바탕으로 같이 일하기 좋은 개발자는 어떤 개발자인지, 어떻게 효율적으로 일할 수 있는지 고민하며 한 층 더 성장할 수 있었습니다.

### 관심사

- 프론트엔드: 웹 클라이언트 개발을 주로 해왔습니다. 성과와 잘못을 바로 확인할 수 있고 유저와 직접 인터렉션 할 수 있는 프론트엔드 개발을 좋아합니다.
- 문서화: 프로젝트 레포지토리의 README를 꾸미는 것, 블로그에 학습한 내용에 대한 포스팅을 남기는 것을 좋아합니다. 제가 이해한 것을 다른 사람에게 잘 설명할 때 보람을 느낍니다.
- 소프트웨어 아키텍처: 소프트웨어 설계와 관련된 서적들을 읽으며 중요한 내용들을 정리하고, 이론들이 실제 소프트웨어 개발에 어떻게 적용될 수 있을지 고민해보는 것을 즐깁니다.

## 지향하는 개발자상

- 지식을 경험과 결합하여 깔끔한 문서를 통해 동료 개발자들에게 인사이트를 줄 수 있는 개발자
- 구도자(求道者)의 태도로 본인의 부족함을 항상 인지하며 좀 더 나은 방법과 방향을 끊임없이 추구하고 배우는 개발자
- 사람들의 삶에 도움이 되는 완성된 결과물을 처음부터 끝까지 만들어 생산할 수 있는, 세상에 도움이 되는 개발자

### [1. 판례요지봇](https://github.com/MaxKim-J/supreme-court-API)

> 프로젝트 기간: 2020.7 ~ 2020.9  
스택: Server(Express.js, postrgreSQL, jest, twitter API), Client(React.js, Redux, Redux Saga)
- 대한민국 법원 법령정보 웹사이트의 대법원 공개 판례를 크롤링하여 하루 3번씩 판례요지의 한 단락을 트윗하는 트윗봇입니다.
- 트위터에 로스쿨 재학생 커뮤니티가 어느정도 형성되어 있다는 점을 인지하여 기획을 진행했습니다.

- 현재 약 260여 명의 유저가 사용 중이며, 2020년 3월 앱 출시 후 기능을 추가해가며 계속 유지보수 중입니다.
